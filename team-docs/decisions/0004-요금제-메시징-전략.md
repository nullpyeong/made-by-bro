# 0004. 요금제 메시징 전략 (앵커링·하루단위·리스크리버설)

| | |
|---|---|
| 상태 | ✅ 채택 |
| 날짜 | 2026-06-21 |
| 파트 | Marketing |
| 결정자 | 김지평 |

## 맥락
학원 베이스 비즈니스라 "돈 내라"는 메시지에 학부모 반발이 크다. 요금제 섹션이
가격을 정면으로 먼저 보여줘서 *비용*이 먼저 인식되는 문제가 있었다.

## 결정
요금제 섹션을 **가격보다 비교기준·위험제거를 먼저** 보여주는 방식으로 재구성한다. 4가지 레버:
1. **앵커링** — 대면 학원 월 15~25만 ↔ 같은 원장 인강 월 16,900 ("같은 원장, 1/10 값")
2. **하루 단위 환산** — 월 16,900 ≈ 하루 약 560원
3. **리스크 리버설** — 무약정 + 환불 보장을 가격보다 위에 노출
4. **결과 프레이밍** — "수강료"가 아니라 "끝까지 완주시키는 비용"

## 근거
- 비용보다 앵커(비교 기준)와 위험 제거를 먼저 보여주면 가격 저항이 낮아진다.
- 기존 CSS 클래스(`.panchor`, `.assure-row`, `.icn-check` 등) 재사용으로 구현 비용 최소.
