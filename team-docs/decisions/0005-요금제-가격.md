# 0005. 요금제 가격 (월/연간/단건 적정 액수)

| | |
|---|---|
| 상태 | 🟡 잠정 채택 (런칭 후 Data 재검증) |
| 날짜 | 2026-06-21 |
| 파트 | Product |
| 결정자 | 김지평 |

## 맥락
ADR 0004(메시징)는 "어떻게 보여줄까"만 정했고, "**얼마가 맞나**"는 근거 없이 월 16,900·단건 66,000이
들어가 있었다. 신규·무명·1인 런칭에서 적정가를 경쟁사 실측과 포지셔닝 근거로 확정할 필요가 있다.
(상세 조사·도출 과정 → `REPORT/product/요금제-가격분석.md`, 비공개)

## 결정
3개 유료 SKU + 무료 체험 + 형제할인 + 런칭 얼리버드로 확정한다.

| SKU | 가격 |
|---|---|
| 무료 체험 | ₩0 (1강 무료) |
| **월 멤버십** (전과목·무약정) | **₩19,900 / 월** (하루 663원) |
| **연간 수강권** (전과목) | **₩179,000 / 년** (월 14,917 환산, 25%↓) |
| **단건 강의** (1코스·~13강·90일) | **₩49,000** |
| 형제 할인 | 2번째 자녀 −30% |
| 런칭 얼리버드 | 첫 100명 평생 ₩12,900 / 월 |

## 근거
- **경쟁사 실측**: 대형 K-12 인강(엘리하이·밀크티·홈런)은 월 5~13만이나 절반 이상이 기기값·전과목·브랜드 프리미엄.
  기기 없는 순수 콘텐츠 tier는 야나두키즈 19,500·밀크티 24개월 27,000. → EPIC 멤버십 **19,900은 이 tier에서 가장 저렴**하면서 "싸구려"로 안 읽히는 지점.
- **전환 제약 = 가격이 아니라 신뢰** → 16,900→19,900의 3,000원 차이는 전환 영향 미미하고, 녹화 콘텐츠는 한계비용≈0이라 저가로 깎을 이유 없음. 16,900은 과소 수익화 + 약한 저가 시그널.
- **연간 179,000 = 최강 앵커**: 동네 영어학원 **한 달치(15~30만)** ≈ EPIC **1년치**. (오프라인 학원비 월 15~30만은 실측 검증됨 → ADR 0004 앵커 보강.)
- **단건 66,000→49,000**: 무명 첫구매 부담↓ + 멤버십 가치 앵커(≈2.5개월치)로 멤버십 유도. 2코스↑/3개월↑이면 멤버십이 이득이 되도록 가격 사다리 정렬.
- **얼리버드**: 후기 0 단계의 진짜 "왜 지금" 레버 + 초기 신뢰자 보상으로 첫 후기·입소문 확보.
- 버린 대안: (a) 16,900 유지 — 과소 수익화. (b) 대형사 추종 고가 — 무명엔 자살. (c) 초저가 999원류 — 품질 신호 훼손·1인 지속불가.

## 결과 / 영향
- **[Marketing/ADR 0004 수정 필요]** 0004 카피가 "하루 560원(=16,900)" → 19,900 채택 시 "하루 700원 이하" 등으로 수정. (16,900 유지 선택 시 0004 불변.)
- **[Design/Eng]** 랜딩 요금제 섹션을 3-SKU(멤버십 hero + 연간/단건 보조)로 재구성. `docs/pages/home.html` 가격·CATALOG·체크아웃 일괄 반영 필요.
- **[Data]** 19,900 vs 16,900, 단건 49,000 vs 66,000은 런칭 후 전환율 A/B로 재검증 → 본 ADR을 "채택"으로 승격 또는 대체.
- **[Sales/CS·Legal]** 무약정·환불·형제할인·얼리버드 평생가 보장을 약관·환불정책에 반영 (ADR 0004 경고와 연결).
- **[Finance]** 토스 정기결제(빌링) 수수료를 멤버십·연간 마진에 반영 (ADR 0003).
