# 0007. 원장 신뢰 콘텐츠 — 사진·신뢰지표·스토리 (임시값)

| | |
|---|---|

| 날짜 | 2026-06-21 |
| 파트 | Content / Design / Marketing |
| 결정자 | (대기 — 실값 제공 후 김지평 확정) |

## 맥락
후기 0개로 런칭하는 단계라 **원장 본인 = 유일한 신뢰 증명**이다(→ [0006](0006-오프라인-60명-시딩.md)의 "검증된 신뢰"). 그런데 랜딩(`docs/pages/home.html`) 원장 소개·약속 섹션의 신뢰 콘텐츠가 비어 있었다:
- 원장 실사 사진(`assets/founder.jpg`) 없음 → 모노그램 "하"로 대체 중
- 신뢰를 숫자로 증명할 지표(경력·학생수·완강률) 없음
- 스토리가 짧음

실명(**김하영**)은 실제 값으로 확정. 사진·신뢰지표·스토리는 **실값이 아직 없어** 페이지가 비어 보이지 않도록 임시값을 넣되, 실값으로 교체될 때까지 **확정하지 않는다**.

## 결정 (제안 — 미확정)
랜딩 원장 콘텐츠를 아래 **임시값**으로 채워 두고, 실값을 받으면 교체한다. 실값 교체·검수 전까지 이 ADR은 `검토중`.

1. **사진** — 임시 플레이스홀더. 실존 인물(원장) 대역이므로 **실제 인물이 아닌 AI 생성 얼굴**(동양인 여성, 차분한 표정)을 사용. 출처 워터마크는 크롭으로 제거, 512px 정사각형. 파일: `docs/pages/assets/founder.jpg`.
   - ⚠️ **실제 원장 사진이 아님.** 런칭 전 실사로 반드시 교체.
2. **신뢰지표** — 숫자만 바꾸면 되도록 `00` 자리표시로 노출. 항목(잠정): 교실 강의 경력 `00년+` / 누적 지도 학생 `00명+` / 정규반 완강률 `00%`. 마크업: `home.html`의 `ul.fstats`.
3. **스토리** — "왜 온라인을 시작했나" 톤의 임시 문단. 실제 원장 스토리 받으면 교체. 위치: 원장 소개 `.fbody > p`(HTML 주석 `임시 스토리` 표시).

## 근거
- **왜 임시라도 채우나**: 빈 신뢰 섹션은 후기 0개 단계에서 가장 큰 약점. 구조·디자인을 미리 잡아두면 실값은 "숫자/문장/사진만 끼우면" 끝.
- **왜 AI 생성 얼굴인가**: 실존 인물(원장)을 다른 실제 사람 스톡 사진으로 대역하면 초상·오인 소지. 실재하지 않는 AI 얼굴이 "임시 플레이스홀더"로 가장 깔끔. 그래도 어디까지나 임시 — 실사 교체 전제.
- **왜 `00` 자리표시인가**: 사용자 요청("00명같이 언제든 채워넣을 수 있게"). 지어낸 숫자를 박으면 거짓 증명이 되므로, **사실인 숫자가 들어오기 전엔 `00`으로 비워 둔다.** 실값 미입력 상태로는 런칭 금지.

## 결과 / 영향
- **[Design/Eng]** `home.html`에 `ul.fstats` 추가, `app.css`에 `.founder .fstats` 스타일 추가. 사진은 기존 `img onerror` 폴백 그대로(파일 없으면 모노그램).
- **[Content]** 사용자가 제공할 실값 3종 대기: ① 원장 실사 1장 ② 신뢰지표 실수치(경력/학생수/완강률 등) ③ 실제 스토리.
- **[Marketing]** 신뢰지표 문구·항목은 0004 메시징 톤과 일치시킬 것(과장 금지, 사실만).
- **리스크**: 임시값을 실값으로 착각하고 런칭하는 것. → 아래 핸드오프의 `유보`로 명시, 실값 교체가 런칭 게이트.

## 사람 확정 핸드오프 (필수)
> ADR은 AI가 임의로 `채택`하지 않는다. `채택`으로 올리기 전에 **반드시 사람이 결정권을 행사**해야 하며, 그 기록을 아래에 남긴다.
> 사람 확정 전에는 상태를 `검토중`으로 둔다.

- **확정자**: (대기)
- **확정일**: (대기)
- **확정 범위**: 실명 `김하영`만 실값 확정. **사진·신뢰지표·스토리는 전부 임시값 — 미확정.**
- **유보 (실값 받으면 교체 후 확정)**:
  - ① 원장 실사 사진 → `assets/founder.jpg` 교체
  - ② 신뢰지표 `00` → 실수치 입력 (또는 항목 자체 교체/삭제)
  - ③ 임시 스토리 → 실제 원장 스토리로 교체
  - ※ 위 3종 교체·검수 완료 전까지 **런칭 금지**, 상태 `검토중` 유지.
