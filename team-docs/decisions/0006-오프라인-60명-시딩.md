# 0006. 오프라인 60명 시딩 — 3개월 콜드스타트 캠페인

| | |
|---|---|
| 상태 | ✅ 채택 (KPI는 측정 시작점, 약속 아님) |
| 날짜 | 2026-06-21 |
| 파트 | Product / Marketing / Data |
| 결정자 | 김지평 |

## 맥락
첫 런칭의 진짜 적은 카니발라이제이션(온/오프 잠식)이 아니라 **무명(콜드스타트)** 이다. 후기 0개·인지도 0인 1인 인강은
"좋은 강의"가 있어도 신뢰 증명이 없어 전환이 안 된다. 한편 오프라인 학원에는 **이미 돈 내고 만족 중인 검증된 고객 60명**이
있고, 이들을 통해 후기·입소문·실측 데이터를 거의 0원에 살 수 있다.

단, 그냥 "온라인 무료"로 풀면 두 가지가 깨진다. ① 부모가 "온라인 공짜면 나 오프라인 왜 돈 내?"(잠식) ②
무조건 무료는 90%가 쓰기만 하고 끝 — 후기·추천·데이터가 안 남는다. 그래서 "무료 배포"가 아니라 **조건이 걸린 3개월 시딩 캠페인**으로 설계한다.

## 결정
오프라인 수강생 60명에게 온라인 복습판을 **3개월 무료**로 열되, "무료"가 아니라 **"오프라인 수강생 전용 복습 혜택"** 으로
포장하고, 혜택의 대가로 **활성화 + 후기**를 교환 조건으로 건다. 추천코드로 외부 유입을 추적하고, 3개월 뒤 실측 CAC를 산출한다.

### 확정된 운영 전제 (2026-06-21 김지평 확정)
| 항목 | 확정 | 이유 |
|---|---|---|
| **대상 범위** | 전원 한 번에 X → **레벨/학년별 2~3개 코호트로 분할 순차** 개방 | 복습 강의 매핑이 레벨별로 다름. 1개 코호트로 파일럿 → 고친 뒤 확산 |
| **무료 기간** | 3개월 (한 학기 호흡) | 짧으면 습관 안 붙고, 길면 잠식 |
| **3개월 후 전환** | 오프라인 **재원생은 계속 무료**(복습 혜택 유지). 이탈/졸업생만 유료 전환 | "다니니까 주는 혜택"의 일관성 = 잠식 안전판 |
| **추천 보상 형태** | 현금/포인트 X → **비현금**(무료기간 연장 or 다음 학기 할인) | 학원에서 현금 리워드는 학부모 정서상 위험("애 팔아 돈?"). 비현금이 깔끔 |

### 교환 조건 (혜택 ↔ 대가)
```
주는 것                          ←→  받는 의무 / 보상 트리거
─────────────────────────────────────────────────────────
온라인 복습판 3개월 무료          ①  등록 후 14일 내 1강 완강 (활성화) — 의무
개인 추천코드 (BRO-홍길동 식)     ②  4강 완강 후 한 줄 후기 1개 — 의무
                                 ③  추천코드로 친구 1명 무료체험 유도 — 보상(무료 1개월 연장)
```
- ①②는 **의무**, ③은 **보상 트리거**(비현금: 무료 1개월 연장).
- **미이행 시 회수 안 함.** 안 지키는 계정은 "비활성"으로 **분류만** 한다. (회수하면 관계가 상해 오프라인 캐시카우까지 흔들림)

### 12주 주차별 실행 (M1 → M2 → M3)
**M1 · 시딩 & 활성화** (적: 무관심)
- **W1** — 코호트1 온보딩. 오프라인 수업 *끝에 원장이 직접* 계정 발급 + "이번 주 복습 여기서". 종이 안내문 1장(QR).
- **W2** — 14일 활성화 데드라인. 미수강자 원장 1:1 넛지. 목표: 코호트 내 **70% 1강 수강**.
- **W3** — 복습 루틴화. "지난 수업 → 온라인 복습 1강" 고정 패턴.
- **W4** — 코호트2 온보딩(W1 반복). 코호트1은 활성 유저로 안착.
- *이 구간 외부 유입·후기 요청 절대 금지* (빈 제품에 추천은 첫인상 자살).

**M2 · 후기 수확 & 추천 점화** (적: 무명)
- **W5** — 활성 유저(≈40명)에게만 **후기 요청**. 폼 1개(별점 + 한 줄). 원장이 직접 요청 → 응답률 폭발.
- **W6** — 추천코드 배포 + 설명. "친구 무료체험 → 너 1개월 연장". 학부모 채널(반모임/맘카페) 자연 노출.
- **W7~8** — 확산 관찰 + 상위 추천인 식별. 후기 **20개** 적재 목표. 누가 입소문 엔진인지 데이터로 확인.

**M3 · 외부 유료 전환 & CAC 실측** (적: 측정 불능)
- **W9** — 후기 20개 깔린 랜딩 외부 공개. 0005 가격 + 얼리버드(첫 100명) 가동.
- **W10~11** — 추천 유입 외부인 → 얼리버드 전환. 무료1강 → 유료 퍼널 측정.
- **W12** — 결산: **실질 CAC = (시딩 운영비 + 추천 보상비) ÷ 외부 유료 전환 수**. 0005 가격을 데이터로 재검증.

### KPI (측정 시작점 — 첫 런칭이라 약속이 아니라 기준선)
| 단계 | 핵심 지표 | 기준선 | 측정 위치 |
|---|---|---|---|
| M1 | 활성화율(14일 내 1강) | 40 / 60 | 수동 체크 or CF Stream 재생로그 |
| M2 | 검증 후기 수 / 추천 가입 / 상위추천인 | 후기 20 | 폼 + 추천코드 표 |
| M3 | 외부 유료 전환 / **실질 CAC** | 전환 30 / CAC 1값 | 토스 결제 + 코드 귀속 |

> `apps/api` 스캐폴딩 전까지는 **스프레드시트 수동 집계로 충분**. 숫자는 약속이 아니라 측정 시작점 — 3개월 뒤 실값으로 본 ADR 갱신.

## 근거
- **콜드스타트 정면 격파**: 0명 vs 60명은 차원이 다름. 후기 0 런칭의 최대 약점을 시작부터 메움.
- **한계비용 ≈ 0의 공짜 시딩**: 온라인 좌석 추가는 서버값뿐이고, 이 60명은 *어차피 온라인을 따로 살 사람이 아님* →
  포기 매출 거의 없음. "마케팅비"보다 더 좋은, 사실상 무손실 시딩.
- **검증된 신뢰의 이전(移轉)**: 이미 만족 중인 고객의 후기라 응답률·신빙성 높음. 사회적 증명을 *돈 주고 살 필요 없이* 보유.
- **포장으로 잠식 회피**: "오프라인 전용 복습 보너스"로 박으면 잠식이 **오프라인 가치 상승(리텐션 강화)** 으로 뒤집힘. 같은 행동, 반대 효과.
- **추천을 "장치"로**: 무료 좌석만 깔면 네트워크 효과는 "희망"에 그침. 추천코드(추적 가능)로 박아야 상위 기여자 식별·CAC 계산이 됨.
- **코호트 분할**: 60명 일괄 개방은 매핑 오류·온보딩 실수가 60명에게 동시 전파됨. 1개 코호트 파일럿으로 리스크를 가둠.
- 버린 대안: (a) 60명 무조건 무료 — 데이터·후기 안 남음. (b) 60명한테도 유료 — 콜드스타트 못 풂, 가장 큰 비대칭 무기 포기.
  (c) 풀 제품 분리(온=콘텐츠/오프=콘텐츠+관리)를 지금 — 첫 런칭엔 과함, "전용 혜택 포장"만으로 충분.

## 리스크 & 안전판
| 리스크 | 조기 신호 | 대응 |
|---|---|---|
| **잠식**(오프라인 "그럼 왜 돈 내?") | 오프라인 문의·이탈 발생 | "오프라인 *전용* 복습 혜택" 포장 강화 + 가격 같은 화면 금지 |
| 활성화 저조 | W2 수강률 < 50% | 원장 강제력 레버(수업 중 넛지) 강화 |
| 후기 안 모임 | W5 응답 저조 | 1:1 직접 요청(원장 신뢰 자산 활용) |
| 추천 안 돎 | W7 가입 0 | 정상. 추천은 보너스, 후기 + 무료1강이 본진 |

## 결과 / 영향
- **[Marketing]** 온라인 가격을 오프라인 가격과 **같은 화면에 절대 안 붙임**. 앵커는 "동네 영어학원 한 달치" 같은 *시장* 가격(0004·0005 정합).
- **[Eng/Design]** 추천코드 발급·추적, 가입 시 코드 귀속, 활성화(수강 시작) 이벤트 로깅 필요. 무료 시딩 계정 tier 구분.
- **[Data]** 측정 항목 확정: 활성화율(수강 시작 %)·평균 수강 강수·후기 수·추천 클릭/가입·상위 추천인·**실질 CAC**.
  KPI는 약속이 아니라 측정 시작점 — 3개월 뒤 본 ADR 갱신 또는 후속 ADR로 재조정.
- **[Content]** 오프라인 진도와 온라인 복습판의 매핑(어느 강의가 어느 주차 복습인지). 코호트별로 다름 — M1 활성화 레버의 핵심.
- **[Sales/CS·Legal]** "3개월 무료 후 재원생 유지 / 이탈생 전환" 안내·약관, 비현금 추천 보상(무료 연장·할인) 지급 정책 명시.
- **알려진 한계**: 이 60명 후기는 *"원장 강의 좋다"*(신뢰)는 강하나 *"인강 단독으로 됐다"*(온라인 단독 증명)는 약함 —
  오프라인을 병행하기 때문. M3 외부 전환자 후기가 그 빈칸을 채울 2차 물결.
- **연결**: 0004(메시징)·0005(가격)와 직결. 미생성된 "온/오프 포지셔닝"은 본 ADR의 "같은 화면에 안 붙임 + 전용 혜택 포장"으로 최소판 흡수.
