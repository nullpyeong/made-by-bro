# 05. Sales/CS

> 영업·문의·환불로 신뢰를 지킨다. 학부모 비즈니스라 응대 하나가 입소문으로 직결된다.

| | |
|---|---|
| 담당 | 김지평 + 원장(누나) |
| 상태 | 정책 수립 필요 |

## 책임 범위
- 결제·환불·수강기간 문의 응대
- 1:1 Q&A 운영 (제품 기능이자 CS 채널)
- 환불 정책 정의·집행
- 응대 템플릿·FAQ

## 왜 초기에 중요한가
환불·문의가 쌓이면 **콘텐츠 만들 시간을 다 잡아먹는다.** 초기 가장 먼저 메꿔야 할 구멍.
→ 정책을 명문화해서 "그때그때 판단"을 없애는 게 핵심.
