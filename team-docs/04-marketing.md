# 04. Marketing/Growth

> 유입을 만들고 퍼널을 뚫는다. 학원은 "발견"이 어렵다 — 좋은 강의도 안 보이면 안 팔린다.

| | |
|---|---|
| 담당 | 김지평 + 원장(누나) |
| 상태 | 운영 중 |

## 책임 범위
- 랜딩 카피·메시징
- 가격 표현(프레이밍) — *가격 정책 자체*는 Product, *표현*은 여기
- 유입 채널 (검색, 입소문, SNS)
- 퍼널 전환 최적화

## 메시징 원칙
신뢰 비즈니스 + 학원 베이스 특성상, **비용보다 비교기준·위험제거를 먼저** 보여준다.
- **앵커링**: 대면 학원 월 15~25만 ↔ 같은 원장 인강 월 16,900
- **하루 단위 환산**: 월 16,900 ≈ 하루 약 560원
- **리스크 리버설**: 무약정 + 환불 보장을 가격보다 위에
- **결과 프레이밍**: "수강료"가 아니라 "끝까지 완주시키는 비용"
