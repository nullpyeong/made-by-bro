# 07. Operations / Finance / Legal

> 정산·세금·약관·개인정보를 굴린다. 안 보이지만 빠지면 사고 나는 영역.

| | |
|---|---|
| 담당 | 김지평 + 원장(누나) |
| 상태 | 정비 필요 (출시 전 필수) |

## 책임 범위
- **Operations**: 결제 정산, 환불 처리 운영, 인프라 비용 관리
- **Finance**: 매출·세금(부가세·사업소득), 정산 주기
- **Legal**: 이용약관, 개인정보처리방침, 전자상거래법 고지

## 왜 출시 전 필수인가
유료 결제를 받는 순간 **전자상거래법·개인정보보호법** 의무가 발생한다.
약관·환불규정·개인정보방침 없이 결제받으면 법적 리스크.

## 지금 결정된 것 (Decided)
- 결제사 토스페이먼츠 → 정산은 PG 정산 주기 따름 ([0003](decisions/0003-결제-토스페이먼츠.md))

## 열린 질문 (Open)
- 사업자 형태 (개인사업자/교습소 명의?)
- 이용약관·개인정보처리방침 초안 (템플릿 기반)
- 청약철회(환불) 규정 — Sales/CS의 환불 정책과 일치시켜야 함
- 미성년자(초·중등) 결제 — 보호자 동의 처리
