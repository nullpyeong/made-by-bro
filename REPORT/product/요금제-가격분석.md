## 0. 한 줄 결론

신규·무명·1인 런칭에서 **전환을 막는 건 가격이 아니라 신뢰**다. 그래서 가격은 "싸게"가 아니라
"믿을 만하면서 부담 없는" 지점에 두고, **무료 1강 + 무약정 + 얼리버드**로 위험을 0으로 만든다.

| SKU | 권장가 | 한 줄 근거 |
|---|---|---|
| 무료 체험 | **₩0 (1강)** | 후기 0을 대체하는 유일한 증명 장치 = 전환 척추 |
| 월 멤버십 (전과목·무약정) | **₩19,900 / 월** | 하루 663원. 야나두키즈(19,500)·밀크티 순수콘텐츠(27,000)보다 낮고, 16,900보다 가치 시그널↑ |
| 연간 수강권 (전과목) | **₩179,000 / 년** | 월 14,917 환산(25%↓). "동네 영어학원 **한 달치**로 EPIC **1년**" 앵커 |
| 단건 강의 (1코스·~13강·90일) | **₩49,000** | 무명 첫구매 부담↓(66k→49k). 멤버십 가치 앵커(≈2.5개월치) |
| 형제 할인 | **2번째 자녀 −30%** | 경쟁사 공통 관행 + 학원 연계 포지셔닝 |
| 런칭 얼리버드 | **첫 100명 평생 ₩12,900/월** | 후기 0 단계의 "왜 지금" + 초기 신뢰자 보상 |

## 1. 경쟁사 실측 (2025~2026, 웹 조사)

가격이 대부분 **태블릿/스마트펜 번들 + 전과목 + 장기약정**이라 그대로 비교하면 안 된다.
"순수 콘텐츠 월 환산"으로 정규화해서 봐야 EPIC(녹화 영어 인강, 기기 없음)과 비교 가능.

| 서비스 | 운영사 | 표기 가격 | 순수 콘텐츠 월 환산(추정) | 비고 |
|---|---|---|---|---|
| **아이스크림 홈런** 초등 | 아이스크림에듀 | 기본 ~6만 / 프리미엄 ~9만, 기기포함 24개월 월 11.8만~12.9만 | **6만~9만** | 전과목+AI+기기 번들 |
| **밀크티** 초등 | 천재교육 | 12개월 월 54,000 / 24개월 월 27,000 | **2.7만~5.4만** | 약정 길수록↓, 태블릿 별도/포함 혼재 |
| **밀크티** 중등 | 천재교육 | 일시불만, 12개월 128만(월 10.7만) | **10만+** | 월납 없음 |
| **엘리하이** 초등 | 메가스터디(대교 아님*) | 월 10만원대 (태블릿+스마트펜+무제한) | **10만+** | 형제할인 월 최대 −28,000 |
| **야나두키즈** | 야나두 | 통합 이용권 월 19,500 / 영어 단과 2년 74,000 | **월 1.9만 / 영어단과 3,083** | 영어 단과는 사실상 최저가 tier |
| **클래스101** (성인 참고) | 클래스101 | 단건 고가, 구독 연 18만(월 1.5만) | **월 1.5만** | 단건보다 구독으로 유도하는 구조 |

\* 엘리하이는 메가스터디교육 브랜드(초등). "대교"는 오인 — 조사 중 정정.

### 오프라인 앵커 (ADR 0004 검증용)
- 초등 동네 영어학원: **주 2~3회 월 15만~30만** (강남·수도권 +10~20%), 등록비 5~10만·교재비 2~5만 별도.
- → ADR 0004의 "대면 학원 월 15~25만" 앵커는 **실측으로 검증됨**. 오히려 보수적(상단 30만까지).

## 2. 시장 포지셔닝 — EPIC은 어디에 서나

```
가격(월 환산)
10만+  ┤ 엘리하이 · 밀크티중등 · 홈런프리미엄      ← 브랜드+기기+전과목 (못 따라감, 따라갈 필요 없음)
 5~9만 ┤ 홈런기본 · 밀크티초등(12개월)
 2~3만 ┤ 밀크티초등(24개월 순수콘텐츠)
 ~2만  ┤ 야나두키즈 통합(19,500)  ★ EPIC 멤버십 19,900  ← 여기
 ~1.5만┤ 클래스101 구독
```

**핵심 통찰**
1. 대형사 가격의 절반 이상은 **기기값·브랜드 프리미엄**이다. EPIC은 기기 없음 → 구조적으로 저렴할 수밖에/있다.
2. EPIC의 진짜 경쟁자는 엘리하이가 아니라 **"인강 결제했다가 안 본 경험"** 그 자체다. → 가격 경쟁이 아니라 "이번엔 끝까지 본다"(완청)로 차별화. ADR 0004 메시징과 일치.
3. 무명 1인은 **대형사보다 싸되, 너무 싸면 "싸구려"** 신호. 야나두키즈 tier(~2만)가 "믿을 만하면서 접근 가능"한 최적점.

## 3. 적정가 도출 — 왜 이 숫자인가

### (a) 월 멤버십 ₩19,900 (현재 16,900에서 인상 권장)
- **전환 제약은 가격이 아니라 신뢰** → 3,000원 차이는 전환에 거의 영향 없음. 그 레버는 무료 1강이 담당.
- 녹화 콘텐츠는 **한계비용 ≈ 0** → 이미 만든 강의는 멤버십이 곧 순마진. 굳이 저가로 깎을 이유 없음.
- 19,900 = **하루 663원**, 심리적 2만원 미만 유지, 야나두키즈(19,500)와 동급 → 신뢰 tier.
- 16,900은 (1) 살짝 저가 시그널 (2) 1인 지속가능성 측면에서 과소 수익화.
- ⚠️ 트레이드오프: ADR 0004 카피가 "하루 560원(=16,900)"으로 박혀 있음 → 19,900 채택 시 "하루 700원 이하"로 수정 필요. (대안: 16,900 유지하고 0004 그대로) — **0005에서 결정.**

### (b) 연간 수강권 ₩179,000 (신규 SKU)
- 월 환산 14,917 = 표준 대비 **25%↓** ("3개월 무료" 체감: 19,900×12=238,800 → 약 6만 절약).
- **최강 앵커**: 동네 영어학원 **한 달치(15~30만)** ≈ EPIC **1년치(17.9만)**. 가격 저항을 정면 격파.
- 현금흐름·리텐션(연 단위 묶기)에 유리. 요금제에서 "가장 인기" 배지 위치.

### (c) 단건 강의 ₩49,000 (현재 66,000에서 인하 권장)
- 무명 강사에게 첫 결제 ₩66,000은 신뢰 대비 과한 요구. 49,000으로 첫구매 장벽↓.
- 단건의 역할은 "한 코스만 원하는 사람"용 + **멤버십 가치 앵커**. 49,000 = 멤버십 ≈2.5개월치 → "2개 이상 들을 거면 멤버십이 이득"이 자연 성립 → 멤버십으로 유도.
- 90일 무제한 복습은 유지(이미 ADR 결정).

### (d) 형제 할인 −30% (2번째 자녀)
- 엘리하이(월 −28,000) 등 경쟁사 공통. EPIC은 "학원 연계·형제 추가 할인"이 이미 포지셔닝에 있음 → 정합.

### (e) 런칭 얼리버드 평생 ₩12,900/월 (첫 100명)
- 후기 0 단계의 **"왜 지금"** 레버 (가짜 마감 대신 진짜 한정).
- 초기 신뢰자에게 평생 최저가 보상 → 입소문·첫 후기 확보 (플라이휠 시동).
- 가격 민감층은 얼리버드가 흡수, 표준가는 건강하게 19,900 유지.

## 4. 단건 vs 멤버십 — 카니발라이제이션 점검

| 구매 의향 | 합리적 선택 | 설계 의도 |
|---|---|---|
| 한 코스만 | 단건 49,000 | OK (소수) |
| 2코스 이상 / 3개월+ | 멤버십 (49k×2=98k > 19,900×4=79,600) | **멤버십 유도** ✅ |
| 장기 (1년) | 연간권 179,000 | **묶기·리텐션** ✅ |

→ 가격 사다리가 "오래 볼수록 멤버십/연간이 이득"으로 자연 정렬됨. 의도대로.

## 5. 미해결 / 후속 (다른 렌즈로 넘김)

- **[Sales/CS·Legal]** 무약정·환불 보장을 마케팅에서 약속 → 실제 환불 규정/청약철회와 일치 필요 (ADR 0004도 동일 경고).
- **[Data]** 19,900 vs 16,900은 결국 A/B 대상. 런칭 후 전환율로 검증 → 0005를 "검토중"이 아닌 "잠정 채택 + 데이터 재검증"으로.
- **[Finance]** 토스페이먼츠 정기결제(빌링) 수수료가 멤버십 마진에 반영돼야 (ADR 0003).
- **[Product]** 단건/멤버십/연간 3-SKU가 인지 부담일 수 있음 → 랜딩은 "멤버십 hero + 단건/연간 보조"로 위계 정리 (마케팅 작업과 연결).
